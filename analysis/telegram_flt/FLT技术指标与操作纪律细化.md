## 总体方法框架

FLT 的技术体系不是单独看一个指标，而是把多个条件叠加：

1. 先判断市场阶段：牛市主升、4浪调整、5浪末端、震荡修复，还是共振下跌。
2. 再判断指数环境：上证是否站稳关键点位，科创50是否共振，创业板是否破位。
3. 再判断板块位置：是否是主线板块，是否处在主升浪，是否有政策/产业催化。
4. 再判断个股形态：趋势是否还在，是否突破箱体，是否缩量回踩，是否跌破关键均线。
5. 最后决定动作：持有、低吸、加仓、做T、减仓、清仓或空仓等待。

他很少用“单指标买卖”。比如 RSI 超买不一定马上卖，要看是否主升浪、是否有量、是否强趋势顶部钝化；跌破20日线也不一定马上卖，要看是温和跌破还是暴跌超卖。

## 均线方法

### 1. 20日线：短线生命线

FLT 对20日线的定位很明确：它是短线趋势是否被破坏的重要分界线。

他的规则：

| 情况 | 判断 | 操作 |
| --- | --- | --- |
| 温和跌破20日线，RSI仍在中位 | 趋势可能开始转弱 | 减仓或离场 |
| 暴跌跌破20日线，RSI极低 | 已经恐慌超卖 | 不急着割，等反弹后处理 |
| 站上20日线且放量 | 短期多头趋势修复 | 可观察5日线买点 |
| 20日线未破 | 趋势仍在，不要提前看空 | 持有为主 |

他在 `message66125` 里把这个逻辑说得很完整：20日线是短线生命线，但跌破方式要区分。温和跌穿、RSI在50附近，应该卖；暴跌跌穿、RSI到个位数，则不该恐慌减仓，而应等反弹。

可还原成操作公式：

```text
如果收盘温和跌破20日线，且 RSI 没有进入极端超卖：
    减仓或止盈
如果快速暴跌跌破20日线，且 RSI 极低：
    不追杀，等反弹到更合理位置再处理
如果20日线没破：
    不因为主观恐慌提前卖掉趋势股
```

### 2. 10日线：趋势持仓和止盈线

10日线在他体系里通常用于强趋势股的中短线持仓。

常见规则：

- 趋势股沿10日线走，可以继续拿。
- 从底部拿到上涨较大后，可以先减一部分，剩余仓位等破10日线止盈。
- 最高点追进去后，如果第二天就跌破10日线，应果断跑。
- 强势股如果跌破10日线，往往说明短线主升节奏变坏。

典型消息：

- `message60787`：趋势股不要恐高，5日线建仓，10日线加仓，破10日线走人。
- `message135552`：从底拿到现在的，可以减 1/3 或 1/2，剩下等破10日均线止盈。
- `message132450`：最高点进，第二天跌破10日线，肯定要跑。

可还原成操作公式：

```text
趋势股低位启动后：
    5日线附近试仓
    10日线附近加仓或持有
    有效跌破10日线，剩余仓位止盈/止损
```

### 3. 5日线：强势股试仓和节奏线

5日线更多用于主升浪强势股。

他对5日线的用法：

- 主升浪里，5日线可作为建仓/低吸点。
- 股价严重脱离5日线，说明短线过热，适合做T或止盈。
- 尾盘跌破5日线，尤其是目标位附近且强弱切换时，可以作为止盈触发。
- 短线强势股如果沿5日线上行，不应过早恐高。

典型消息：

- `message60787`：5日线建仓，10日线加仓，破10日线走人。
- `message50889`：突破6%后回落是卖点，因为股价严重脱离五日线。
- `message150640`：目标位后，如果华虹公司转弱，尾盘 2:55 跌破5日线全部止盈。

可还原成操作公式：

```text
如果个股处于主升浪且未明显过热：
    回踩5日线可试仓
如果短线大涨并明显脱离5日线：
    不追高，可考虑做T或止盈
如果已到目标位附近，且尾盘跌破5日线：
    视为强转弱信号，执行止盈
```

### 4. 周线/月线：判断更大级别空间和风险

周线/月线在他体系里用于判断大周期，不是日内买卖。

常见用法：

- 周线箱体突破，代表更大级别空间打开。
- 月线缩量回踩趋势线，没有有效跌破，说明中期趋势仍可观察。
- 周线收在关键均线下方，会影响后续第二波行情判断。
- 个股若进入月线下跌趋势，不应再想着“火中取栗”。

他对大周期的态度是：大周期决定能不能重仓拿，短周期只决定加减节奏。

## RSI 方法

### 1. RSI > 80：超买，但不是无脑卖

FLT 把 RSI 80 以上视为超买区，尤其适合震荡市做日内T。

规则：

- RSI 超过80，说明短线过热。
- 震荡市里，RSI>80 是较好的高抛区。
- 如果行情特别强，可能出现顶部钝化，不能简单因为超买就卖光。
- 个股若不是“无压力位主升”，多数 RSI 80+ 后会回落。

典型消息：

- `message50888`：RSI超过80就是超买，在这个区间卖出，日内做T成功率较高；但行情好时可能顶部钝化。
- `message60006`：除非个股走出没有压力位的主升，否则60分钟RSI 80+ 多数会回落，是短线卖点。

可还原成操作公式：

```text
如果 RSI > 80 且市场是震荡行情：
    可高抛/做T
如果 RSI > 80 但个股处于强主升且放量突破：
    不急于清仓，观察是否顶部钝化
如果 RSI > 80 后冲关键压力失败：
    视为高抛或减仓信号
```

### 2. 顶部钝化：强势和风险同时存在

顶部钝化在他的语言里不是单纯看空，而是“强势过热，需要修复”。

他的理解：

- 60分钟、120分钟 RSI 顶部钝化，说明短线涨太快。
- 如果不是无压力位主升，钝化后大概率回踩。
- 强主升可以顶着钝化继续涨，但需要量能和趋势支撑。
- 钝化修复通常通过横盘或回调完成。

典型消息：

- `message70778`：60分钟、120分钟全部顶部钝化，没有好买点，只能按主升浪走，5日线建仓、10日线加仓、跌破走人。
- `message122266`：若能接受周五、周一向下共振跌一次修复RSI指标，可以不动。

可还原成操作公式：

```text
如果短周期 RSI 顶部钝化：
    不追高
    等回踩5日/10日线，或等钝化修复
如果已有底仓且趋势没破：
    可持有，接受短线回落修复
如果钝化叠加冲压力位失败：
    减仓或做T
```

### 3. RSI 极低：不是割肉点，通常等反弹

他明确反对在 RSI 极低时恐慌杀跌。

规则：

- RSI 到个位数或极端超卖，是恐慌区。
- 暴跌跌破均线后，如果 RSI 极低，先等反弹。
- 反弹到 RSI 50-60 区间，再考虑减仓，胜率更高。

典型消息：

- `message66125`：暴跌跌穿20日线、RSI到个位数，不该减仓，要等反弹。
- `message68024`：哪怕不看好后市，等 RSI 从超卖反弹到60附近再减仓，也能回血。

可还原成操作公式：

```text
如果暴跌后 RSI 极低：
    不在最低情绪点割肉
    等技术反弹
如果 RSI 从超卖修复到50-60：
    再考虑减仓或调仓
```

## MACD 与背离

### 1. 顶背离：严肃风险信号

相比 RSI，FLT 提到 MACD 次数少，但对“顶背离”的态度非常明确：顶背离要跑，至少要减仓。

典型消息：

- `message62307`：MACD顶背离一定要跑，不跑也要减仓。
- `message78455`：上证在3月2日、3月3日出现顶背离，调整远未完成。
- `message64414`：顶背离后跌穿20日线，如果反弹后再有效跌破20日线，风险更大。

可还原成操作公式：

```text
如果出现 MACD 顶背离：
    先降低仓位
如果顶背离后又跌破20日线：
    确认趋势转弱，继续减仓
如果顶背离后只是横盘而不是大跌：
    说明指数强，但调整时间可能拉长
```

### 2. 底背离：下跌末端的观察信号

他在解释5浪下跌时提到，浪5末端可能出现底背离：价格创新低，但 RSI/MACD 不再创新低，说明下跌动能衰竭。

这个信号在他体系里不是马上满仓，而是“寻找底部和反弹机会”的辅助信号。

可还原成操作公式：

```text
如果价格新低，但 RSI/MACD 不创新低：
    观察是否进入下跌末端
    等量能、支撑、均线修复确认
```

### 3. 金叉/死叉：辅助趋势判断

他偶尔提到纳斯达克周线死叉、DIFF周线金叉、EXPMA金叉等。整体看，他不把金叉死叉当独立交易信号，而是用来解释市场情绪和趋势阶段。

用法：

- 周线死叉会让散户不敢追，反而可能成为主力洗盘背景。
- 周线金叉可作为大级别启动前的领先信号。
- EXPMA金叉后若叠加 RSI 钝化，仍可能需要调整修复。

## 量能方法

### 1. 量能代表分歧程度

FLT 对量能的核心理解是：放量代表分歧大，缩量代表分歧小。

规则：

- 放量：买卖双方激烈交换，可能是突破，也可能是出货。
- 缩量：抛压减轻，筹码稳定，适合观察回踩确认。
- 关键位置放量不过，容易冲高回落。
- 突破关键压力位，理论上需要放量。

### 2. 突破必须看量，回踩最好缩量

这是他最完整的一套技术规则。

突破阶段：

- 突破箱体、前高、关键压力位，最好放量。
- 放量代表主力有意愿解放套牢盘。
- 如果冲压力位但没有补量，容易失败。

回踩阶段：

- 健康回踩应该缩量。
- 缩量说明抛盘衰竭，筹码锁定好。
- 放量回踩说明抛压还没释放完，可能是假突破或出货。

典型消息：

- `message50884`：要突破关键压力位，必然是放量，更不用说新高。
- `message92497`：放量回踩往往意味着抛压未释放干净，缩量回踩才是健康确认。
- `message56717`：摸到4150还不放量，就会冲高回落；补量则看震荡。

可还原成操作公式：

```text
突破前高/箱体：
    需要放量确认
突破后回踩：
    需要缩量确认
如果突破时无量：
    谨慎，可能冲高回落
如果回踩时放量：
    谨慎，可能假突破或出货
```

### 3. 成交额用于判断赚钱效应

他会用市场总成交额判断行情温度。

规则：

- 成交额从高位大幅缩量，可能意味着恐慌消化，下一波赚钱效应将开启。
- 放量恐慌盘之后，次日不一定还需要大幅放量才能反弹。
- 市场持续无量，容易产生恐慌和震荡。
- 成交额要结合证券、权重、科技等板块轮动看。

## 突破、回踩、支撑、压力

### 1. 突破箱体后的标准动作

他的标准模型：

1. 长期横盘形成箱体。
2. 放量突破箱体顶部。
3. 回踩箱体顶部。
4. 原压力位变成支撑。
5. 缩量回踩不破，则可加仓或持有。
6. 跌回箱体，突破失败。

典型消息：

- `message50835`：完全突破箱体且突破后没有回踩，后续回踩0.5分位并出现变盘信号，可上一部分仓位试。
- `message60001`：股价突破箱体、60分钟RSI顶部钝化后，会回踩箱体压力位，看是否转成支撑。

可还原成操作公式：

```text
如果放量突破箱体：
    不急追，等回踩确认
如果回踩箱体顶部缩量不破：
    可试仓/加仓
如果跌回箱体内部：
    视为突破失败，降低仓位
```

### 2. 压力位不是点位，是资金行为测试区

FLT 反复提到 4100、4130、4150、4197、4200、4300 等点位，但他的使用方式不是机械点位，而是观察资金是否能放量通过。

判断规则：

- 盘中过一次不算站稳，收盘站稳才更有效。
- 到压力位如果缩量，容易冲高回落。
- 过压力位后如果量化/趋势资金跟进，可能加速。
- 压力位附近若卖盘很多但主力承接，可能为后续突破蓄力。

典型消息：

- `message122266`：想要站稳4197，必须收盘站稳，不是盘中过一次。
- `message72615`：4120-4130能否突破是重点；如果突破，资金会再走出赚钱效应。

### 3. 破位后不要火中取栗

对已经破位的股，他态度非常明确：不应该继续研究怎么火中取栗。

规则：

- 已经破位且进入月线下跌趋势，优先回避。
- 跌破关键均线后如果反弹无力，应离场。
- 空头趋势股不如回踩均线的多头趋势股。

典型消息：

- `message138278`：一个已经破位、决定走月线下跌的股，不应该研究怎么火中取栗。
- `message58447`：买回踩五日线的多头趋势股，好过买不知道何时回箱体顶部的空头趋势股。

## 做T 方法

### 1. 做T只适合有执行力的人

他多次强调，做T对盘中判断和执行要求很高，不适合所有人。

要求：

- 能看关键时间点。
- 能判断量价是否匹配。
- 能在冲不过关键涨幅/压力位时卖出。
- 能接受卖飞风险。
- 能盘后复盘，先模拟练习。

典型消息：

- `message50891`：做T对盘中决策和执行能力要求很强，刚开始可以用复盘工具先练。

### 2. 日内关键时间：10点、10点半

他提到日内做T要观察两个时间段：

- 10:00
- 10:30

观察内容：

- 股价是否在这些时间点拉升。
- 拉升时量价是否匹配。
- 新高是否缩量。
- 是否冲关键涨幅失败。

典型消息：

- `message50874`：观察10点和10点半，若股价拉升但量价不匹配，新高缩量，基本可以做T。

可还原成操作公式：

```text
10:00 或 10:30 股价拉升：
    如果新高缩量：
        高抛做T
    如果放量突破关键位：
        不急卖，观察延续
```

### 3. 看 3%、6% 等日内涨幅关口

他会用 3%、6% 这类日内涨幅作为压力测试区。

规则：

- 股价冲到 3% 附近，突破不了，可高抛。
- 冲到 6% 附近，突破不了，可高抛。
- 突破6%后重新跌回来，是较简单的卖点。
- 如果市场无量、震荡，冲关失败的卖飞概率较低。

典型消息：

- `message50875`：看是否突破3%、6%；若打到5.8%、5.4%附近过不去，应及时卖出。
- `message50889`：早盘突破6%后重新跌回来，就是卖点。

### 4. RSI > 80 可辅助做T

日内做T中，RSI>80 是他常用的辅助卖点。

规则：

- 震荡市 RSI>80 可以高抛。
- 强行情可能钝化，不能一刀切。
- RSI 超买叠加缩量冲高，更适合卖。

### 5. 不要把长线股频繁做飞

他对做T的矛盾态度很明显：他认可做T，但更担心散户做T做飞主升浪。

规则：

- 主升浪早期和中期，频繁做T容易卖飞。
- 长线目标明确时，不做T、死拿到目标位可能更好。
- 做T适合降低成本，不适合破坏主线仓位。

典型消息：

- `message97294`：有些长线股只要死拿到目标位即可，做T反而可能错过30-40%收益。
- `message119817`：散户常被洗一次，回本赚几个点就止盈做T，最后赚不到大钱。

## 仓位管理

### 1. 核心仓位 + 短线仓位

FLT 有一套清晰的分层仓位思想：

- 核心仓位：放在大主线、大指数、确定性更高的方向。
- 短线仓位：用于题材、个股、手痒交易。

典型配置：

- `message54247`：50% 科创50 ETF 长线基本盘 + 50% 仓位跟题材短线。

他的逻辑：

- 牛市要想赚大钱，必须有大仓位压在主线上。
- 但全部做短线，容易卖飞、踏空。
- 全部做长线，可能手痒或资金效率低。
- 所以用核心仓位保证主线收益，用小仓位参与短线。

### 2. 仓位大小取决于目标空间

他倾向于在看到大目标位时堆大仓位。

典型消息：

- `message147860`：看到目标位能涨50%或100%，就尽可能堆大仓位，然后死拿；涨多了、快到压力位就买少点，到了目标位直接清了止盈。

可还原成操作公式：

```text
如果目标空间大、逻辑强、位置低：
    可提高仓位
如果已接近压力位或短线涨幅大：

如果达到目标位：
    分批或直接止盈
```

### 3. 没把握就用指数ETF

典型消息：

- `message148443`：没把握就把总仓位放在科创50里。
- `message148445`：开仓几百只股，可能连满仓指数ETF都跑不赢。

### 4. 调整期短线先撤，长线看周期

他在2月中旬就说过：如果3月有风险，短线先不玩，长线继续拿。

规则：

- 看不清跌多少时，不必全部卖。
- 但短线仓位可以先撤，避免回吐。
- 长线仓位若是产业周期主线，不因小跌调仓。
- 若进入指数共振调整，短线应空仓等待。

典型消息：

- `message50739`：不知道到底跌多少，至少看到风险，短线仓位先不玩；长线拿周期，不因小跌调仓。

## 止盈、止损、减仓

### 1. 时间窗口到了，宁愿少赚也不要回吐

他多次强调：阶段风险出现时，宁愿少赚，不要把前面利润吐回去。

典型消息：

- `message50705`：3月想带大家减一部分仓位，短线先不玩，宁愿少赚也不要回吐。
- `message50709`：2月尾、3月头冲到4300，就开始减仓。

### 2. 到目标位就执行

他对目标位的纪律较强：

- 到目标位附近开始减。
- 超过目标位后若强弱切换，尾盘破5日线可全部止盈。
- 冲下一个压力位失败，也可以清仓。

典型消息：

- `message150640`：华虹到目标位后，若中芯补涨而华虹不跟，尾盘跌破5日线全部止盈。
- `message145206`：反转过后去下一个压力位，所以第二天冲10%可以清仓。

### 3. 分批止盈

分批逻辑：

- 从底部拿上来、涨幅较大时，可先减 1/3 或 1/2。
- 剩余仓位用10日线或5日线作为止盈线。
- 如果仍在主升，不要一次性清得太早。

典型消息：

- `message135552`：底部拿到现在的，可减 1/3 或 1/2，剩下等破10日均线止盈。

### 4. 清仓条件

他会清仓的情况：

- 个股达到或超过目标位，且出现转弱信号。
- 主线板块已经兑现大幅收益，后续风险收益比下降。
- 指数进入共振调整，买什么都难赚钱。
- 个股破位且进入更大级别下跌。
- 最高点追入后次日跌破10日线。

典型消息：

- `message154247`：本轮半导体主升浪兑现后，要求剩余半仓华虹、沪硅、中芯、科创50ETF 全部清掉。
- `message132450`：最高点进、第二天跌破10日线，肯定要跑。

### 5. 止损不是一刀切，要区分买点和位置

他并不是机械止损派。判断是否止损，要看：

- 是不是主线板块。
- 是不是低位长线逻辑。
- 有没有破关键均线。
- 是温和破位还是恐慌暴跌。
- RSI 是否极端超卖。
- 是否已经到目标位。

他的隐含规则：

```text
买在高位：
    破10日线要跑
买在低位主线：
    短期浮亏不一定止损，重点看目标位和趋势
恐慌暴跌：
    不在 RSI 极低处割肉
温和破位：
    减仓/止损
```

## 长线 vs 短线

### 1. 长线的核心：拿住主线到目标位

FLT 反复说，A 股大收益往往来自少数强赚钱效应阶段。如果在主升浪里频繁短线，很容易卖飞。

长线规则：

- 找低位、主线、目标空间大的股。
- 忍受途中回撤和洗盘。
- 不因大盘短跌卖掉逆势强股。
- 到目标位或趋势破坏再处理。

典型消息：

- `message78455`：很多股必须从头拿到尾，才会发现能拿到大利润；中间可能涨50%、回吐20%、再涨30%、再跌15%。
- `message97294`：找到低位、从924以来没涨太多的股，目标位明确，就死拿到目标位。

### 2. 短线的核心：只在趋势和量能支持下做

短线规则：

- 做短线要看市场阶段，不能在共振调整里硬做。
- 短线买点优先回踩5日线、突破回踩确认。
- 短线卖点看 RSI>80、冲关键涨幅失败、缩量新高、跌破10日线。
- 短线仓位不能影响核心长线仓位。

### 3. 他反对“既想长线又天天波段完美”

他认为很多人最大的问题是：明明拿的是长线逻辑，却每天想规避每一次下跌；明明做短线，又不执行止损。

可还原成原则：

```text
做长线：
    接受波动，目标位和趋势没坏前不乱动
做短线：
    严格执行，跌破条件就走
不要用短线情绪管理长线仓位
不要用长线借口拖延短线止损
```

## 组合成一套执行流程

### 开仓前检查

1. 是否是当前市场主线？
2. 指数是否处于可操作阶段？
3. 个股是否多头趋势？
4. 是否靠近5日/10日/20日线或突破回踩位？
5. 是否有足够目标空间？
6. 是否已经远离5日线、处于追高区？
7. 买入后失效条件是什么？

### 持仓中检查

1. 是否跌破5日线、10日线、20日线？
2. 跌破是温和跌破，还是暴跌超卖？
3. RSI 是高位钝化、正常修复，还是极端超卖？
4. 是否放量冲不过压力位？
5. 是否缩量回踩支撑不破？
6. 板块主线是否还在？
7. 指数是否进入共振下跌？

### 减仓/止盈检查

1. 是否到目标位？
2. 是否接近重要压力位？
3. 是否 MACD 顶背离？
4. 是否温和跌破20日线？
5. 是否从底部涨幅很大，可先减 1/3 或 1/2？
6. 是否剩余仓位可用10日线跟踪止盈？
7. 是否板块主升浪已兑现，风险收益比下降？

### 空仓/等待条件

1. 指数共振调整，买什么都跌。
2. 顶背离后调整未完成。
3. 大部分主线股破位。
4. 个股进入月线下跌趋势。
5. 市场无量震荡且没有明确主线。
6. 自己无法执行做T或短线纪律。

## 需要警惕的执行误区

1. 把 RSI>80 当成无脑清仓信号。FLT 会结合主升浪和量能判断。
2. 把跌破20日线当成机械止损。FLT 会区分温和跌破和暴跌超卖。
3. 把突破当成追高理由。他更偏好突破后的缩量回踩确认。
4. 把做T当成低风险技巧。他认为做T对执行力要求很高，容易做飞。
5. 把长线股按短线恐慌卖掉。他强调主线长线要拿到目标位。
6. 把短线亏损用长线逻辑安慰。他对高位买入后跌破10日线的处理很果断。
7. 开太多杂股。他更偏好主线ETF或少数核心主线。

## 一句话总结

FLT 的技术体系可以压缩成一句话：主线趋势股低位重仓，沿5日/10日线做节奏，20日线看短线趋势生死，RSI/MACD判断过热和背离，放量突破后等缩量回踩确认，目标位和破位纪律到了就执行。

本整理只是对聊天记录中的观点进行结构化复盘，不代表这些方法正确，也不构成投资建议。
